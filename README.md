# Video-with-diverse-styles

> 浏览器内的视频调色工具，主打 LUT 预设滤镜 + 抽卡随机生成，无需专业知识即可一键美化或玩出意外视频。

**核心特性：**
- 📹 支持 MP4 / MOV / WebM 视频上传（拖拽或点击）
- 🎨 三类预设体系：经典风格 / 氛围风格 / 脑洞风格
- 🎲 抽卡系统 — 随机颜色通道反转、色相错乱、RGB错位，制造惊喜
- ⚡ WebGL 实时预览，无需等待
- 📦 FFmpeg.wasm 本地处理（规划中），保护隐私，无需后端
- 💾 输出视频支持 MP4 (H.264/H.265) 和 WebM

## 技术栈

- **框架**: Next.js 16 (App Router)
- **样式**: Tailwind CSS v4
- **视频处理**: HTML5 Video + Canvas/WebGL
- **导出**: FFmpeg.wasm (规划中)
- **部署**: Cloudflare Pages / Vercel

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run start
```
