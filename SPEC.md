# Video-with-diverse-styles MVP 需求文档

**版本：** v0.2  
**日期：** 2026-04-07  
**状态：** 开发中

---

## 一、产品概述

**产品名称：** VideoStyle  
**产品定位：** 面向普通用户的在线视频调色工具，主打 LUT 预设滤镜 + 抽卡随机生成，让用户无需专业知识即可一键美化或玩出意外视频。  
**核心价值：** 浏览器内完成视频调色，本地处理保护隐私，无需安装软件。

---

## 二、目标用户

- 有视频调色需求但不具备专业知识的普通用户
- 短视频创作者、vlogger、社交媒体内容生产者
- 对视频美感有追求但不想学习 DaVinci Resolve 等专业工具的人群
- 追求趣味性、愿意分享"意外效果"的用户

---

## 三、核心功能（MVP 范围）

### 3.1 视频上传
- 支持拖拽或点击上传
- 支持格式：MP4、MOV、WebM
- 限制：单个文件最大 500MB（后续可扩展）
- 上传后显示视频缩略图和时长

### 3.2 预设体系

预设分为三类，满足不同使用场景：

| 类型 | 说明 | 示例风格 |
|------|------|----------|
| **经典风格** | 常见、稳妥 | 电影感、暖调、冷调、复古、日系 |
| **氛围风格** | 情绪向、氛围感 | 赛博朋克、昭和日记、阴间滤镜、马戏团、莫兰迪 |
| **脑洞风格** ⚠️ | 大胆、实验性 | 负片、单色反转、色相错乱、RGB错位、像素抽离 |

**UI 呈现：**
- 经典/氛围：以缩略图网格展示
- 脑洞：标注"⚠️ 实验性"，吸引好奇心
- 点击预设后 WebGL 实时预览，无需等待

### 3.3 抽卡系统

独立于预设之外的随机生成模块，制造惊喜感和传播点。

**颜色抽卡：**
- 随机反转 R/G/B 其中一个通道的值
- 随机交换两个颜色通道
- 随机偏移色相 30°-180°
- 结果不可预测，每抽一次都是"惊喜"

**参数骰子：**
- 对比度、饱和度、曝光随机组合（范围可极端化）
- 极端参数往往产生意外的"车祸现场"效果
- 用户可以接受这种失控感 → 发朋友圈的动机

**UI 设计：**
- 独立的「🎲 抽卡」按钮
- 点击有专属动画反馈（参考游戏抽卡效果）
- 抽卡结果可再次"抽"或"锁定"

### 3.4 视频导出
- 输出分辨率：同原视频（最高 1080P）
- 输出格式可选：
  - MP4 (H.264) — 推荐，兼容性最好
  - MP4 (H.265) — 体积更小
  - WebM (VP9) — 无版权顾虑
- 导出进度条展示
- 完成后支持浏览器直接下载

### 3.5 辅助功能
- 播放/暂停、进度条拖拽
- 视频音量控制
- 全屏预览模式

## 四、技术方案

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端框架 | Next.js 16 (App Router) | React 生态 |
| UI 组件 | Tailwind CSS v4 | 快速搭建，界面美观 |
| 视频预览 | HTML5 Video + WebGL | LUT 实时预览（规划中） |
| 视频处理 | FFmpeg.wasm | 浏览器内处理视频（规划中） |
| 抽卡逻辑 | 纯前端 JS | 随机颜色矩阵生成 |
| 静态部署 | Cloudflare Pages / Vercel | 免费额度，CDN 加速 |

**数据流向：**
```
用户视频 → 浏览器内存 → WebGL预览 → FFmpeg.wasm处理 → 浏览器下载
           （全程本地，无服务器上传）
```
